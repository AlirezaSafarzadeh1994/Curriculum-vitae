## How to publish with GitHub Pages

### Option 1: Personal website URL
Use this if you want the URL to be:

```text
https://AlirezaSafarzadeh1994.github.io/
```

1. Create a new GitHub repository named exactly:

```text
AlirezaSafarzadeh1994.github.io
```

2. Upload all files in this folder to the root of that repository.
3. Commit the files to the `main` branch.
4. GitHub Pages should publish automatically. If it does not, go to:

```text
Settings → Pages → Build and deployment → Deploy from a branch → main / root
```

### Option 2: Project website URL
Use this if you want a URL like:

```text
https://AlirezaSafarzadeh1994.github.io/cv/
```

1. Create any repository name, for example `cv`.
2. Upload all files in this folder to the root of the repository.
3. Go to:

```text
Settings → Pages → Build and deployment → Deploy from a branch → main / root
```

## How to edit

- Update text content in `index.html`.
- Update colors and layout in `styles.css`.
- Replace the PDF in `assets/` if your CV changes.
- Replace `assets/og-card.svg` if you want a custom social preview image.
